Для збірки проекту був використаний інструмент GULP.  
Було написано файл gulpfile.js, за допомогою якого
GULP виконував task-и. Також використано деякі 
плагіни для GULP:
	-  rename (для переіменування файлів)
	-  concatcss (збирає всі .css файли в один)
	-  minifycss (мініфікую всі .css файли)
	-  notify (для виведення повідомлення про закінчення
				виконання роботи GULP-ом)

Номер версії проекту знаходиться у напівзгенерованому 
файлі  package.json:

"version": "1.0.0"
